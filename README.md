# ssd1306---dspic33fj128mc802
Librería para controlar pantallas OLED SSD1306 (128x64) desde el microcontrolador dsPIC33FJ128MC802. Incluye primitivas gráficas (líneas, círculos, texto)  Optimizada para I2C a 400kHz y algoritmos clásicos de computación gráfica. Perfecta para proyectos embebidos que necesitan interfaz visual.
